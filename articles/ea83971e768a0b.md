# SPA とは

SPA とは、Single Page Application の略で、1 つのページでアプリケーションを実装することを指します。
モダンな JS フレームワークを使っている人は、なじみ深いと思います。

簡単に言うと、HTML は１つのみで、ページ遷移は JavaScript で行うということです。
